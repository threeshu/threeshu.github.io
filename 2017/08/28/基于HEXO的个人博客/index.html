基于HEXO的个人博客

发表于 2017-08-28 | 分类于 笔记

Hexo官方定义

  • Hexo是一个快速、简洁且高效的博客框架；
  • Hexo使用Markdown（或其他渲染引擎）解析文章；
  • 在几秒内，即可利用靓丽的主题生成静态网页。

    本地安装Hexo的教程

    安装Git

  • 注册GitHub账号，打开git bash使用$ ssh-keygen -t rsa -C "youremail@example.com"创建SSH Key;
  • 登陆GitHub，打开“account setting”->”ssh keys”->”add ssh key”,填上任意title，在Key文本框里粘贴id_rsa.pub文件的内容，点“add key”，你就可以看到已经添加的key。

    安装Node.js

    安装Hexo

  • 使用命令$ npm install -g hexo-cli进行安装；
